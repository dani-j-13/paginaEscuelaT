Ciclo Superior: Modalidad de Informática

1. Laboratorio de Hardware

Descripción:

El taller de Laboratorio de Hardware está orientado al aprendizaje práctico y profundo de los componentes físicos que componen un sistema informático. Los estudiantes aprenderán a identificar, montar, desarmar y reparar distintos dispositivos de hardware. El taller está diseñado para proporcionar a los alumnos una comprensión sólida de cómo funcionan los equipos desde el interior, trabajando directamente con componentes como procesadores, tarjetas madre, memorias RAM, discos duros y periféricos.

Objetivos:

  • Conocer y manejar los distintos tipos de hardware de un computador.
  • Realizar instalaciones y mantenimiento de hardware.
  • Diagnosticar y reparar fallos comunes en equipos informáticos.
  • Aprender a ensamblar un equipo desde cero.

Herramientas necesarias:

  • Destornilladores, pinzas, y otros utensilios de taller.
  • Componentes de computadoras como placas base, tarjetas de expansión, y memorias.
  • Equipos de diagnóstico (multímetros, programas de diagnóstico de hardware).

Puntos a saber:

  • Funcionamiento de los componentes internos de un computador.
  • Cómo realizar diagnósticos de fallas y reparaciones.
  • Instalación de dispositivos periféricos.
  • Ensamblaje completo de un equipo informático.

2. Laboratorio de Sistemas Operativos

Descripción:

En este taller, los estudiantes aprenderán a instalar, configurar y administrar diversos sistemas operativos, incluyendo Windows, Linux y macOS. Además, aprenderán sobre la gestión de archivos, la configuración de redes, la gestión de usuarios y seguridad. El taller enfatiza el entendimiento profundo de cómo los sistemas operativos interactúan con el hardware y cómo optimizar su rendimiento para tareas específicas.

Objetivos:

  • Aprender a instalar y configurar sistemas operativos.
  • Comprender cómo funcionan los sistemas operativos a nivel de kernel y software.
  • Gestionar y configurar redes dentro de un sistema operativo.
  • Administrar permisos y usuarios.

Herramientas necesarias:

  • Computadoras con acceso a diversos sistemas operativos.
  • Herramientas de particionamiento de discos.
  • Software de virtualización como VirtualBox o VMware para probar diferentes sistemas operativos sin alterar los equipos físicos.

Puntos a saber:

  • Instalación de sistemas operativos en diferentes plataformas.
  • Configuración y gestión de la red en sistemas operativos.
  • Resolución de problemas comunes en la administración de sistemas.
  • Uso avanzado de la terminal o línea de comandos, especialmente en Linux.

3. Programación

Descripción:

El taller de Programación está diseñado para enseñar a los estudiantes los fundamentos de la programación, así como las mejores prácticas para escribir código limpio, eficiente y fácil de mantener. Los alumnos aprenderán a programar en lenguajes populares como Python, Java y C++, desarrollando habilidades para crear aplicaciones desde cero, realizar depuración y optimización de código.

Objetivos:

  • Aprender los fundamentos de la programación y algoritmos.
  • Desarrollar habilidades en lenguajes de programación como Python, Java y C++.
  • Diseñar y desarrollar proyectos de software.
  • Comprender las estructuras de datos y algoritmos.

Herramientas necesarias:

  • Computadoras con IDEs (Integrated Development Environments) como Visual Studio Code, Eclipse, o PyCharm.
  • Herramientas de depuración y control de versiones (Git y GitHub).
  • Acceso a bases de datos y APIs para prácticas avanzadas.

Puntos a saber:

  • Lógica de programación y resolución de problemas mediante código.
  • Diseño de interfaces de usuario simples.
  • Uso de estructuras de datos como listas, pilas, colas y árboles.
  • Desarrollo y pruebas de software.

4. Laboratorio de Aplicaciones

Descripción:

En el Laboratorio de Aplicaciones, los estudiantes desarrollarán y aprenderán a mantener aplicaciones informáticas, tanto para escritorio como para la web. El taller abarca desde la creación de aplicaciones simples en entornos como Visual Studio hasta el diseño de aplicaciones complejas con bases de datos y comunicaciones en red. Los estudiantes aprenderán sobre el ciclo de vida de una aplicación, desde el diseño inicial hasta la implementación y el mantenimiento.

Objetivos:

  • Desarrollar aplicaciones de software tanto de escritorio como web.
  • Gestionar bases de datos dentro de las aplicaciones.
  • Comprender el ciclo de vida de una aplicación, incluyendo las fases de pruebas, implementación y mantenimiento.
  • Trabajar en proyectos de desarrollo de aplicaciones reales.

Herramientas necesarias:

  • IDEs como Visual Studio, Eclipse, o Android Studio para desarrollo de aplicaciones.
  • Bases de datos como MySQL, SQLite o PostgreSQL.
  • Herramientas de desarrollo web (HTML, CSS, JavaScript, y frameworks como React o Angular).

Puntos a saber:

  • Desarrollo de aplicaciones completas, desde el diseño hasta la implementación.
  • Gestión de bases de datos en aplicaciones.
  • Diseño de interfaces de usuario y experiencia de usuario (UI/UX).
  • Integración de servicios web y APIs para crear aplicaciones más dinámicas.
